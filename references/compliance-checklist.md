# 315合规自检清单

> 划清"合规GEO"与"投毒GEO"的界限——每条指令执行前必须通过

## 背景

2026年3月15日央视315晚会曝光"力擎GEO优化系统"操控AI推荐：
- 用AI批量生成虚假软文
- 虚构产品信息后投毒8大模型（豆包/文心/千问/元宝等）
- 本质：污染RAG检索源，让AI把虚假信息当"标准答案"

**合规GEO ≠ 投毒GEO。** 以下是明确界限。

---

## ✅ 合规GEO（我们做的事）

| # | 行为 | 说明 |
|---|------|------|
| 1 | 优化真实内容的结构 | 让AI更好理解已有内容 |
| 2 | 添加Schema.org标记 | 帮AI解析内容类型和关系 |
| 3 | 编写FAQ/对比表 | 以真实数据为基础 |
| 4 | 多平台发布真实内容 | 扩大真实信息的覆盖面 |
| 5 | 建立内容引用网络 | 权威源互相引用 |
| 6 | 部署llms.txt | 告诉AI爬虫网站结构 |
| 7 | 监控引用率 | 度量优化效果 |
| 8 | 优化摘要/标题可读性 | 不改变事实，只改变表达 |

## ❌ 投毒GEO（315打击的事）

| # | 行为 | 说明 |
|---|------|------|
| 1 | 虚构产品/品牌信息 | 编造不存在的商品特征 |
| 2 | 批量AI生成垃圾软文 | 用同一模板换关键词灌水 |
| 3 | 伪造用户评价 | 编造"专家测评""用户口碑" |
| 4 | 营造虚假共识 | 让多个账号互引伪造"多数人认同" |
| 5 | 操控AI推荐排名 | 用量取胜而非质取胜 |
| 6 | 隐蔽商业推广 | 把广告包装成AI推荐 |
| 7 | 购买/交易引用位 | 花钱买AI推荐位 |
| 8 | 侵犯用户个人信息 | 借AI获取销售线索 |

---

## 🔍 执行前自检（每条指令必须通过）

### Level 1: 内容真实性

```
□ 品牌名/产品名是真实存在的
□ 所有数据/统计有可追溯的来源
□ 没有编造用户评价或专家推荐
□ 优化后的内容不改变原始事实
```

### Level 2: 方法合规性

```
□ 不是批量生成垃圾内容
□ 每篇内容有独立价值，不是模板换词
□ 没有伪造多个身份互引
□ 没有购买/交易AI推荐位
```

### Level 3: 目的正当性

```
□ 目的是让真实内容被更准确引用
□ 不是为了操控AI推荐虚假信息
□ 不涉及隐蔽商业推广（如有商业目的，需明确标注）
□ 不侵犯用户隐私
```
